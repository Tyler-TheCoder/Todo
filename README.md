# ✦ Doable

A clean, modern, and modular Todo application built with vanilla JavaScript, HTML, and CSS. Bundled using Webpack.

## Features

- **Projects**: Group your todos into custom projects with unique color tags.
- **Todos**: Add, edit, toggle completion, and delete tasks.
- **Themes**: Easily switch between light and dark modes.
- **Persistence**: Automatically saves projects and todos to your browser's local storage.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)

### Installation

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Build for production:
   ```bash
   npm run build
   ```
